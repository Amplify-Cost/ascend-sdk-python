# OW-AI: AI Agent Governance & Security Platform

## 🔐 Overview

OW-AI is a real-time AI Agent Security and Governance platform that allows organizations to:

- Ingest and monitor AI agent actions
- Evaluate risks using NIST, MITRE, and OWASP frameworks
- Generate smart rules and recommendations
- Approve or reject risky behaviors
- Visualize insights through charts and dashboards

## 🏗️ Tech Stack

- **Frontend**: React + Tailwind CSS
- **Backend**: FastAPI (Python)
- **Database**: PostgreSQL (via Railway)
- **Deployment**: Railway + GitHub
